## 各系统状态视觉说明

**Loading 系统**：
- IDLE/CHECKING：全屏纯色背景（#ECEFF1）
- LOADING：背景 + Logo + 进度条 + "资源加载中……"文字
- LOADING_ERROR：保持当前进度 + 重试按钮（红色描边）
- TRANSITIONING：背景淡出动画

**主界面系统**：
- 背景：全屏背景图，浅色渐变，轻微动态装饰元素
- Logo：游戏标题"围住Boss"，居中顶部，卡通风格字体，金色/暖色
- 开始按钮：橙→红渐变圆角矩形，白色粗体文字，下方小号关卡提示
- 排行榜入口：圆形奖杯图标，64×64px（触控扩展至 88×88px）
- 设置入口：圆形齿轮图标，64×64px
- 分享入口：右上角分享图标，带红点提示（首次分享）
- 道具栏：底部横向排列四个 64×64px 道具图标和持有数量

**核心玩法系统**：
- 网格：N×N 方格，交点可点击，已放置障碍物显示棕色/灰色图标
- Boss：红色系 Q 版角色，位于网格交点上
- 障碍物：玩家放置为木栅栏（棕色），地图固定为石墩（灰色）
- 状态颜色：可行走路径浅色，不可行走深色
- 步数指示器：右上角显示"已用/上限"

**关卡结算系统**：
- 胜利界面：暖色（金色/橙色）氛围，Boss 鼻青脸肿表情，大号金色胜利文案
- 失败界面：冷色（灰色/蓝色）氛围，Boss 得意表情，浅灰色失败文案
- 统计面板：半透明卡片背景，显示步数和耗时数据

**道具系统**：
- 可用状态：彩色图标 + 数量数字（白色）
- 不可用状态：彩色图标 + 右上角"+"号
- 耗尽状态：灰色图标 + 50% 透明度
- 获取弹窗：半透明遮罩 + 居中弹窗卡片

**排行榜系统**：
- 弹出层：右侧滑入，半透明遮罩 + 白色面板（500×700px）
- 标签页：三个维度水平排列，选中态下划线/高亮
- 列表项：排名序号 + 头像（微信） + 昵称 + 成绩
- 自我排名：背景高亮（浅蓝色）

**设置系统**：
- 弹出层：左侧滑入，半透明遮罩 + 白色面板
- 开关滑块：64×32px，圆角 16px，绿色(开)/灰色(关)
- 版本号：底部小号灰色文字

**分享系统**：
- 分享卡片：游戏名称 + 关卡截图 + 邀请文案（由微信原生 API 处理）
- 红点提示：分享按钮右上角小红点，直径 8px
